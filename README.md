# tlb-simulation
This project explores the performance of different Translation Lookaside Buffer (TLB) replacement algorithms under a variety of simulated memory access scenarios. The goal is to understand how algorithm design and workload characteristics impact TLB efficiency.

# how to run 
1. Open the Python file
2. Run the program (e.g., press ▶ Run or use: python filename.py)
3. View the TLB simulation results printed in the console
Note: The program uses randomness, so results may vary slightly each run.
